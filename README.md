# Motor_RaspberryPi
ロボットシステム学の授業で製作したデバイスドライバを参考にした２つのモータを制御するデバイスドライバです。

# 動作環境
|OS|Ubuntu Server 20.04.3 LTS|
|---|---|
|ハードウェア|Raspberry Pi 3|
|モータ|タミヤ　3速クランクギヤーボックスセット ×2|
|モータドライバ|東芝　TB6643KQ ×2|
